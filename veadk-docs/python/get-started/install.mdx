---
title: "安装"
---

您可以使用如下方式安装 VeADK：

<CodeGroup>
    ```bash pip
    pip install -U veadk-python
    # Requires Python 3.10+
    ```

    ```bash uv
    uv add veadk-python
    # Requires Python 3.10+
    ```
</CodeGroup>

您可以通过如下方式安装扩展包：

<CodeGroup>
    ```bash pip
    pip install -U veadk-python[extensions]
    # Requires Python 3.10+
    ```

    ```bash uv
    uv add veadk-python[extensions]
    # Requires Python 3.10+
    ```
</CodeGroup>

此外，您还可以通过如下方式安装 VeADK 的特定分支：

<CodeGroup>
    ```bash pip
    pip install git+https://github.com/volcengine/veadk-python.git@[branch-name]
    # Requires Python 3.10+
    ```

    ```bash uv
    uv add git+https://github.com/volcengine/veadk-python.git@[branch-name]
    # Requires Python 3.10+
    ```
</CodeGroup>

您还可以使用 VeADK 提供的基础镜像：

| **镜像** | **镜像地址** |
| -------- | -------- |
| 稳定版 | `veadk-cn-beijing.cr.volces.com/veadk/veadk-python:latest` |
| 主分支预览版 | `veadk-cn-beijing.cr.volces.com/veadk/veadk-python:preview` |
| 指定版本号 | `veadk-cn-beijing.cr.volces.com/veadk/veadk-python:[version]` |

当您成功安装后，请参阅[快速开始](quick-start.md)来构建您的第一个 AI Agent。
